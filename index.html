색상 계열 일관성 강화:
v5.0은 같은 top1 안에서도 색이 완전히 다른 계열로 점프했음 (예: 개방성 → 보라/마젠타/라벤더/페리윙클이 섞임).
v5.1은 top1 = 색 계열 결정, top2 = 같은 계열 안의 명도/채도 변주 구조로 통일. 따라서 사용자가 자신의 결 색상을 봤을 때 "이 색은 주로 어떤 성향이구나"를 직관적으로 파악할 수 있음.
