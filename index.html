색상 계열 일관성 강화:
v5.0은 같은 top1 안에서도 색이 완전히 다른 계열로 점프했음 (예: 개방성 → 보라/마젠타/라벤더/페리윙클이 섞임).
v5.1은 top1 = 색 계열 결정, top2 = 같은 계열 안의 명도/채도 변주 구조로 통일. 따라서 사용자가 자신의 결 색상을 봤을 때 "이 색은 주로 어떤 성향이구나"를 직관적으로 파악할 수 있음.

5. 변경되지 않은 것 (유지)

항목상태비고
colorComboDesc (20개 조합 설명문)유지의미적 표현이라 색상 변경과 무관 ("창의적 아이디어를 현실로 만드는" 등)
채점 로직 / 점수 계산유지색상은 시각적 요소만 — 영향 없음
SLOT_COUNT, AXIS_MEAN/STD, 가중치유지색상은 백엔드 무관
매칭 타입 P1~P7유지로직 무변경
파동 주파수/진폭/위상 공식유지공식 자체 무변경, 색상만 새 팔레트로 렌더링
traitLevelDescriptions (성향별 설명문)유지텍스트 콘텐츠 변경 없음

6. 영향받는 UI 영역

영역적용 방식
결 해석 모달 — 개인 카드 (♂/♀)대표 색상 도트, 색상명, 게이지 바 색상 모두 새 팔레트
결 해석 모달 — 사분면 다이어그램의 점 색상♂/♀ 점 위치는 그대로, 색상만 새 팔레트로
결 해석 모달 — 두 사람의 궁합 카드아이콘 배경 그라데이션이 두 사람의 새 색상으로
결 해석 모달 — "결 읽는 법" 색상 카탈로그 (12개 미리보기)12개 카드 모두 새 팔레트로 동기화
리포트 헤더의 프로필 동그라미 (남/여)각 성향에 맞는 새 팔레트 컬러로
파동 캔버스 (Relationship Resonance)두 사람 파동의 그라데이션이 새 팔레트로 그려짐
레이더 차트 (8각형 스펙트럼)그대로 (남=파랑/여=핑크 고정색이라 변경 무관)

☐색상 이름이 "딥 라일락", "민트 그린", "탠저린" 등 새 이름으로 표시
☐"결 읽는 법" 펼침 → 색상 카탈로그 12개가 모두 새 팔레트
☐여러 차례 랜덤 테스트 → 다양한 색상 조합이 자연스럽게 나타남
☐우호성 1순위 결과와 정서안정 1순위 결과가 시각적으로 구분 가능 (인접 톤이지만 차이 인지)
☐구 색상 이름 ("사파이어", "에메랄드", "코럴 레드" 등)이 어디에도 표시되지 않음
관찰 포인트: 우호성(#44B8AE)과 정서안정(#42B8CE)이 매우 가까운 청록 계열이라, 두 축이 함께 강한 사용자(예: top1=우호, top2=정서)의 경우 색상 그라데이션이 부드럽게 연결됨. 반대로 top1과 top2가 멀리 떨어진 축이면(예: 개방+외향) 보라→코랄로 변화감이 강한 그라데이션. 이 자연스러운 색상 흐름이 v5.1의 핵심 시각 개선점.
